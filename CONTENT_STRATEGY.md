## Brand Voice Guidelines

### Tone Characteristics
- **Calm authority**: Confident without being aggressive
- **Editorial clarity**: Precise, well-structured sentences
- **Deeply human**: Professional but approachable
- **No jargon**: Clear, accessible language
- **Measured confidence**: Never performative or hyperbolic

### Example Phrases Aligned with Brand Voice
✅ "Strategic recruitment marketing expertise"
✅ "Fractional model gave us strategic depth"
✅ "No retainers, no overhead—just results"
✅ "Measured confidence; never performative"
✅ "Clear problem → solution framing"

❌ Avoid phrases like:
- "Game-changing solutions"
- "Revolutionary approach"
- "Best-in-class" (overused)
- "Cutting-edge" (jargon)
- "Next-level" (vague)

## Mobile Content Optimization

### Mobile-Specific Adjustments
- Sticky CTA button always visible on mobile
- Compressed spacing (20-30% reduction)
- Fluid typography scaling via clamp()
- Hamburger menu for navigation
- Full-width buttons on mobile
- Touch targets minimum 44px

### Mobile Performance
- Lazy-load images below fold
- Minimize initial bundle size
- Defer non-critical scripts
- Optimize for 3G/4G networks

## Accessibility Content Guidelines

### Plain Language
- Average sentence length: 15-20 words
- Avoid complex terminology
- Use active voice
- Short paragraphs (3-4 sentences max)

### Scannable Content
- Bullet points for lists
- Clear section headings
- Visual hierarchy with whitespace
- Descriptive link text (no "click here")

### Screen Reader Optimization
- Semantic HTML structure
- ARIA labels where needed
- Skip navigation link
- Descriptive button text
- Form field labels properly associated
